# MERN-URL-Shortener
# 🔗 Full-Stack URL Shortener (MERN)

A clean, responsive, production-ready MERN stack application (**MongoDB, Express, React, Node**) to shorten long URLs and redirect traffic seamlessly.

---

## 🚀 Core Features

* **Instant URL Shortening:** Maps long destination URLs to secure, compact codes.
* **Auto-Redirection Engine:** Node server handles paths and forwards users immediately.
* **Responsive System UI:** Fluid, clean design optimized for mobile and desktop screens.
* **Strict Data Validation:** Robust backend validation to prevent malformed or broken links.

---

## 🛠️ Technology Stack

* **Frontend:** React.js, Vite, Tailwind CSS, Axios
* **Backend:** Node.js, Express.js
* **Database:** MongoDB Atlas (via Mongoose ODM)
